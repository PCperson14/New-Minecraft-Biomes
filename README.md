# New-Minecraft-Biomes
This is a Mod for NeoForge Minecraft version 1.21.8. **THIS MOD WILL NOT WORK ON FORGE AND FABRIC!**
Put the mod in to
```
C:\Users\(Your Username)\Appdata\Roaming\.minecraft\Mods
```
Now all you need to do is load up Minecraft!
If you want to change the biome to single biome. The only custom biome currently is palm.
New update 1.0.1 bug fixes and allows mobs to spawn in the Palm biome.

**Possible Errors**

1. Not using NeoForge. If you don't use NeoForge the code it compiled to only use NeoForge no others.
2. Not using Minecraft 1.21.8. The mod is coded to only work wth 1.21.8.
3. Other mods in the mods folder that are not for NeoForge or Minecraft 1.21.8.
4. Using Bedrock Edition. If you are using Bedrock Edition this mod won't work becuase it is coded to work with Java Edition.
5. Outdated version of Java. If you have not updaded your Java version it a while that can cause all kinds of issues.
6. Look at the wiki. If you are still getting errors look at the wiki and forums they have more infomation.
